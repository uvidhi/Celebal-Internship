# Week 2 - E-Commerce Sales SQL

## Overview

This assignment focuses on designing and working with an E-Commerce Sales database using MySQL. The database was created from scratch, sample data was inserted, and different SQL queries were written to understand and practice important database concepts.

The assignment covers SQL basics, filtering, aggregation, joins, relationships, transactions, and other advanced SQL concepts.

## Objectives

* Create an E-Commerce Sales database.
* Design tables using primary keys and foreign keys.
* Insert sample data into the database.
* Practice SQL queries from basic to advanced level.
* Understand how relationships and transactions work in MySQL.

## Topics Covered

### Section A – SQL Basics

* SELECT statements
* DISTINCT
* Primary Keys
* Constraints
* INSERT statements
* CHECK, UNIQUE and NOT NULL constraints

### Section B – Filtering and Optimization

* WHERE clause
* ORDER BY
* LIMIT
* LIKE
* BETWEEN
* IN
* Indexes

### Section C – Aggregation

* COUNT()
* SUM()
* AVG()
* MIN()
* MAX()
* GROUP BY
* HAVING

### Section D – Joins and Relationships

* INNER JOIN
* LEFT JOIN
* RIGHT JOIN
* Foreign Key relationships
* Understanding table relationships

### Section E – Advanced Concepts

* CASE statement
* ACID properties
* Transactions (START TRANSACTION, COMMIT, ROLLBACK)
